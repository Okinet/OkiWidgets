With OkiBarSlider you can build slider with any html that you want. You can use ul/li structure or divs with complex content from editor like TinyMCE - all is up to you!

Basic usage

Note that OkiBarSlider is cloning slides at initialization for continous effect. You can change this behaviour by cloneAtFrontAndEnd settings field (if you see 'holes' just increase value).

Slide #01
Slide #02 - little longer
Slide #03 - very long text

Custom navigation

You should use custom navigation when you already have buttons for each slide. By default OkiBarSlider creates links automatically.

Slide #01
Slide #02 - little longer
Slide #03 - very long text
button for first slide
button for second slide
button for third slide

FadeIn and FadeOut version

OkiBarSlider can work also as fade slider. If you set fadeInsteadSlide setting field to true the cloneAtFrontAndEnd field will be internally setted to 0.

Slide #01
Slide #02 - little longer
Slide #03 - very long text

Paginator version

OkiBarSlider can work also as paginator. You need to overwrite some default css. This mode is usefull then you don't know what dimension will content have.
