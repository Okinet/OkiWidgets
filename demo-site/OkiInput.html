If you need to customize browser's input you can use OkiInput. It wraps real browser's input with html structure that you can easly style to get proper result. OkiInput supports also place holder builded on span witch is much more flexible than just placeHolder attribute.

Basic usage

After initialization OkiInput adds class found in "data-css-class" attribute. Remember - real input is wrapped by some structure. Class from "data-css-class" will be added to root of that structure not the real input. That's why we need to put it in "data-css-class" instead of "class" attribute.
