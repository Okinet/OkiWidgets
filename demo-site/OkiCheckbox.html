With OkiCheckbox you can customize browser's checkbox widget. OkiCheckbox by default looks after real checkbox for two tags: A and SPAN
- A tag is custom checkbox icon
- SPAN tag is custom checkbox label

NOTE: All classe names and css code can be changed! All is up to you!

Basic usage

In this example checkboxes was wrapped by div.basic-checkbox-row with fixed height. Instead of this you can of course add div with clear: both. Checkboxes have build in default image (inside css's data:image/png;base64) so they are just ready to use!

Synchronizing with real browser's widget

This example shows how to synchronize OkiCheckbox with real browsers widgets. Note that option hideReal was set to false to show real widgets. If you click to OkiCheckbox it automatically changes real widget. But opposite thing may happen - real widget can be changed by some JavaScript code. OkiCheckbox, for performance reasons, doesn't check for that. You shoud manually invoke sync() method on checkboxes that your're sure that was changed.
