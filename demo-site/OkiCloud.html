With OkiCloud you can create box that works as could. Only one cloud can be displayed at once. If you click outside cloud it will dissapear. Clicking inside cloud has no effect.

NOTE: All classe names and css code can be changed! All is up to you!

Multiple clouds

You can of course run multiple instances of the cloud and add some css to have really cloud that don't pull rest of the page content. All is up to you.

Note that cloud parent div has "position: relative" and cloud div has "position: absolute" with "top: Xpx" value equal to parent cloud div height. Because of that cloud will appear under gray box.
