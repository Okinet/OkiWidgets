OkiScrollArea is solution for scrolling any kind of content without using browser's scroll bars. This OkiWidget can be also controlled by touch so you can use it on mobile devices.

Basic usage

Important - OkiScrollArea ommits css width and height. Widget works on dimension passed during initialization.
If you really need to set dimension using only css you should wrap OkiScrollArea div by another div and set dimension to wrapper (by default viewport is set to 100% so scroll area gets wraper sizes)
