OkiScrollBar displays bar with handle that can be used to control other widgets. Position is normalized to <0.0, 1.0> range.

Basic usage

In example below you can see basic usage of the widget. Depending on initialization you can get vertical bar (default) or horizontal bar. Dimension of the bar is controlled via css width and height property.

Vertical OkiScrollBar
Horizontal OkiScrollBar

Configuration examples

Next examples are showed on horizontal orientation. You can of course do the same in vertical orientation.

You can set starting position and size using 'position' and 'size' settings field.

If you want to bigger handle you can do it in css. Handle has class 'osb-handler'

Events

Widget after bar position change call onChange callback if defined.

Move bar with mouse!

API

You can fetch API object after init like in any other OkiWidget by calling OkiScrollBar('api')
