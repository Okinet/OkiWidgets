OkiSelect replaces real browser's select. Single and multiple version is supported. You can also use keyboard for searching option inside list.

Synchronize with real select

If option selection was changed inside real select you need to call sync method. In below example real select was forced to be visible. OkiSelect hides it inside div with zero height and overflow hidden - <select> itself remains visible. It is important because some browsers don't fire key events on hidden form widget. OkiSelect uses that event for searching.

Styling - basic

If you need to change apperance of the OkiSelect just add you own css class and override default rules. Class can be added by two ways:

  • in options passed at OkiSelect initializations
  • in data attribute at html <select> tag
